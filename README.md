#### 初始化项目
```
npm init -y
```
#### 安装hardhat
```
npm install --save-dev hardhat
```
#### 创建hardhat项目，创建typescript project
```
npx hardhat 
```
#### 编译项目
```
npx hardhat compile
```
#### 推送代码到git仓库
```
git remote add origin https://github.com/Mrzhang007/fund_me.git
git branch -M main
git push -u origin main
```
